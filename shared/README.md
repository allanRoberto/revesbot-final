# Shared

`shared/` concentra código reutilizável entre aplicações, separado por linguagem.

- `shared/python/`: utilitários e módulos reutilizados por apps Python.
- `shared/javascript/`: base reservada para código reutilizável entre apps Node.js.

Observação:
- o repositório ainda possui duplicações legadas em alguns apps;
- nesta etapa, o objetivo é consolidar a estrutura sem forçar uma extração ampla.
